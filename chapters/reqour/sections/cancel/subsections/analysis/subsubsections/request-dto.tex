The cancel request DTO is JSON of the following format:

\begin{lstlisting}[numbers=none]
{
  "taskId": "string",
  "callback": {
    "method": "string",
    "uri": "string"
  }
}
\end{lstlisting}

where both fields have the same meaning as in Cloning DTO described in \ref{subsubsec:cloning-request-dto}.
